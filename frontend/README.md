# Payout Engine Frontend

This is the React 19 dashboard built to interface with the Payout Engine backend.

## Tech Stack
* React 19
* Vite 6
* Tailwind CSS v4
* Axios for API calls

## Local Development
To run this frontend locally without Docker:
```bash
npm install
npm run dev
```

The application will be available at `http://localhost:5173`. Make sure your Django API is running simultaneously.
